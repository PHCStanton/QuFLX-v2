Here are **practical, high-impact AI prompts** you can copy-paste directly into the **Ask AI** feature in QuFLX v2. They are designed to be concise, context-rich, and optimized for getting the most accurate, actionable trading advice — especially for your low-frequency A+ OTC binary style (92%+ payout, 1:2 R/R, 5–10% daily DD cap).

All prompts assume the current chart screenshot + live context (asset, timeframe, candles, indicators, regime) is automatically sent — so you only need to paste the question.

### 1. Pre-Trade / Alert Evaluation (Most Used – Highest Edge)
```
Analyze this current alert and chart:
[ paste the full alert text here, e.g. "Strong momentum trending detected on AUDNZD_OTC 1m – ADX rising, price above EMA-16 & BB upper expansion" ]

Give me a short, clear breakdown:
1. Regime (strong momentum, pullback, ranging, breakout, reversal)
2. Key confluences (EMA, RSI, BB, ADX, candle structure)
3. Optimal entry timing (wait for close? seconds left?)
4. Risk level (1–10) + main danger
5. My historical win rate on this exact setup (from journal. If available)
6. Final call: Enter / Wait / Skip + one-sentence reason
```

### 2. Mid-Trade Reassessment (Protect Profits / Cut Early)
```
I entered [call/put] on [asset] [timeframe] [X seconds] ago.
Current chart + indicators attached.

Quick reassessment:
- Still valid? Yes/No + why
- Regime shift risk now?
- Stay in, early exit, or double down?
- Rough probability of ITM at this moment
```

### 3. Post-Trade Debrief (Fastest Learning Loop)
```
Last trade: [call/put] on [asset] [timeframe] – [won/lost]
Chart at entry attached.
Journal note: [paste your quick tag/note, e.g. "Entered early, RSI 62"]

Analyze:
- Did I follow my rules?
- Biggest mistake (if any)?
- Historical win rate on this exact setup?
- One clear rule I should add or change
```

### 4. Session Planning / Regime Outlook
```
Current session: [London / NY overlap / etc.]
Chart + indicators attached.

Quick outlook:
1. Current regime + probability of transition in next 30–60 min
2. Top 2–3 assets right now (92%+ payout) with highest edge for my style
3. Max trades left today given current drawdown [X%]
4. One thing I should do differently this session (from my journal patterns)
```

### 5. Alert Filtering / Rule Tuning
```
I want only A+ alerts from now on.
My proven edge:
- Win rate >60% on pullbacks to EMA-16 in uptrend
- Avoid last 10s of candle
- Prefer London/NY overlap
- Only 92%+ payout

Rewrite my alert rules to match this exactly.
Suggest 3–4 ultra-specific conditions that should trigger notifications.
```

### 6. Risk & Sizing Guidance (Compounding Focus)
```
Current account: [balance]
Daily drawdown so far: [X%]
Today’s wins/losses: [Y wins – Z losses]
Risk per trade I'm comfortable with: 2–3%
Next alert: [paste alert text]

Calculate:
1. Recommended stake size for 1:2 R/R
2. Max trades left before hitting 10% DD
3. If I lose next 2 → account impact
4. Conservative vs aggressive sizing recommendation
```

### 7. Quick Reality Check / Regime Scan (Anytime)
```
One-sentence regime summary + biggest current risk on this chart.
Rate tradeability 1–10 right now.
```

### 8. Quick Predict (Ultra-Fast Bias)
```
FAST PREDICT for [asset] [timeframe].
Bias: [Call/Put/Neutral] (Confidence %)
Primary Trigger: [Logic]
Expiry: [Target]
Limit response to 3 precise lines.
```

### Power Prefix (Use on Almost Any Question)
Start most prompts with this one line to force maximum context:
```
Using current chart screenshot, latest indicators, regime label, payout assets, session time, and my journal history, answer clearly and concisely:
```

### Quick Reference – When to Use Which Prompt

- Just got an alert → Use #1 (Pre-Trade / Alert Evaluation)  
- Already in a trade → Use #2 (Mid-Trade Reassessment)  
- After win/loss → Use #3 (Post-Trade Debrief)  
- Start of session → Use #4 (Session Planning)  
- Too many alerts → Use #5 (Alert Filtering)  
- Sizing question → Use #6 (Risk & Sizing)  
- Quick gut check → Use #7 (Reality Check)
